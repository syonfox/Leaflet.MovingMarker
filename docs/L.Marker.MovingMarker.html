Class: MovingMarker

MovingMarker(latlngs, durations, options)

new MovingMarker(latlngs, durations, options)

Creates a new MovingMarker object which extends L.Marker. It animates the movement of the marker along a given set of coordinates (latlngs) with a corresponding set of durations (durations) between each coordinate.
Parameters:
Name Type Description
latlngs Array.<L.LatLng> Array of LatLng objects representing the path that the marker should follow.
durations Array.<number> Array of durations (in milliseconds) representing the time that the marker should take to move from one coordinate to the next.
options Object Options for the MovingMarker object.
Properties
Name Type Attributes Default Description
autostart boolean <optional>
 false Whether to start the animation automatically after the object is created.
loop boolean <optional>
 false Whether to loop the animation back to the start after reaching the end.
